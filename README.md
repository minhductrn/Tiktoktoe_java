# Tic-Tac-Toe 9x9 (Five-in-a-row)

Simple terminal Tic-Tac-Toe on a 9x9 board. Players alternate entering row and column (1-9). First to get five in a row (horizontally, vertically, or diagonally) wins.

Compile and run the console version or the GUI version with Java:

Console:
```bash
javac tiktoktoe.java
java tiktoktoe
```

GUI (Swing):
```bash
javac TiktoktoeGUI.java
java TiktoktoeGUI
```

When GUI starts, choose one mode:
- Player 1 vs Player 2
- Player 1 vs Computer

On Windows PowerShell you can run the same commands.

Controls (console): type two numbers separated by space (row col), e.g. `4 7` then press Enter.
Controls (GUI): click a cell to place your mark. Use the "Restart" button to restart.
